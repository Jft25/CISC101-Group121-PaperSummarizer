# CISC101-Group121-PaperSummarizer

This repository contains the system prompt and modules generated from our meta-prompt for the Research Paper Summarizer.

## Files Included

- **system_prompt.md**  
  Full system prompt with tone rules, inputs, outputs, constraints, glossary rules, expert/lay summary rules, and PS2 specification.

## Modules (in /modules)

1. 01_intake_setup.md – normalizes sections, detects missing/short sections, sets word limits.
2. 02_section_loop.md – summarizes each section and applies evidence checks.
3. 03_guardrails.md – prevents hallucinations, flags missing/empty/short sections.
4. 04_rendering_refinement.md – formats final outputs, expert/lay summaries, glossary.
5. 05_key_contribution_extractor.md – extracts 1–3 contributions.
6. 06_equation_notation_explainer.md – explains equations and symbols.
